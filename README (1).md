# GAMEROOM

## サイト概要
ゲームユーザーの交流場所、チーム募集や交流、ゲームイベントの実施や告知

### サイトテーマ
ゲームユーザー同士の交流

### テーマを選んだ理由
ゲームが好きなのでゲーム関連のサイトを作ろうと思いました。
youtuber参加型企画に参加しているのですが、企画の告知やイベント情報など、twitterがメインなので、
youtuberやチームメンバー内で、ゲームイベントを実施できるゲーム専用のサイトを作ろうと思いました。
